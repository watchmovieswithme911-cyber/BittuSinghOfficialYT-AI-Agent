## 🐛 Troubleshooting

**Deployment failed?**
- Check [Actions tab](https://github.com/watchmovieswithme911-cyber/BittuSinghOfficialYT-AI-Agent/actions) for error logs
- Ensure files are committed to `main` branch
- Verify GitHub Pages is enabled in Settings

**Site not loading?**
- Wait 2-3 minutes after push
- Hard refresh your browser (Ctrl+Shift+R)
- Check if GitHub Pages is enabled in repository Settings

**Custom domain?**
- Add CNAME file to root: `bittu-ai-agent.example.com`
- Configure DNS records at your registrar
- See [GitHub Pages docs](https://docs.github.com/en/pages/configuring-a-custom-domain-for-your-github-pages-site)
